# frontend/components

โฟลเดอร์สำหรับ Frontend-specific Components ที่ใช้เฉพาะ UI layer

**หมายเหตุ:** Shared components ยังอยู่ที่ `src/components/` ตามเดิม
โฟลเดอร์นี้รองรับ components ที่จะสร้างใหม่ในอนาคต

| โฟลเดอร์ | หน้าที่ |
|---|---|
| layout/ | Layout components เฉพาะ frontend |
| cards/ | Card components |
| forms/ | Form components |
| upload/ | Upload UI components |
